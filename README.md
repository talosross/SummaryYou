## About this repository

This repository is the official home of **Summary You** ([com.talosross.com](https://github.com/talosross/SummaryYou)).

> **Project history:**
> - **Summary You** was the original project by [talosross](https://github.com/talosross).
> - **Summary Expressive** (by kid1412621/NanoNova) was forked from Summary You and introduced a more modern, efficient, and feature-rich codebase, including Material 3 Expressive UI integration.
> - In 2026, the codebase of Summary Expressive was merged back into this repository to provide the best experience and maintain the original app identity. All references and branding have been restored to "Summary You".
